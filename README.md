# ShareMedia - Content Sharing Platform

A modern web application where users can upload and share videos, pictures, and other media content.

## Features

- ✅ User Authentication (Register/Login)
- ✅ Upload Images & Videos
- ✅ Personal Feed
- ✅ User Profile
- ✅ Responsive Design

## Tech Stack

**Frontend:**
- React 18
- React Router
- Axios
- CSS3

**Backend:**
- Node.js
- Express
- MongoDB
- Mongoose
- JWT Authentication
- Multer (File uploads)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- MongoDB (local or cloud)
- npm or yarn
